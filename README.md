# 📦 Inventory App Management

A real-time **Inventory Management System** built with **Java (Android Studio)** for the mobile front-end and **PHP/MySQL** for the backend.  
Developed for **AutoDiesel Store** to streamline stock tracking, low-stock alerts, and historical reporting — ensuring accurate and efficient inventory control.

---

## 🚀 Features

- 🔄 **Real-time stock tracking** — monitor stock-in and stock-out in real-time  
- ⚠️ **Low-stock alerts** — automatic notifications when inventory is running low  
- 📊 **Historical reports** — view sales and stock movement for decision-making  
- 🔗 **Android–MySQL integration** — seamless communication using PHP REST APIs  
- 🎨 **User-friendly interface** — clean and intuitive UI for better user experience  

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-------------|
| Frontend | Java (Android Studio) |
| Backend | PHP |
